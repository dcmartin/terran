.. title:: Welcome to Terran

Welcome to Terran's documentation!
==================================

**Terran** is a human perception library that provides computer vision
techniques and algorithms in order to facilitate building systems that interact
with people.

You can find the library's source code in `its Github repository
<https://github.com/nagitsu/terran>`_.
